Chat TTS Reader Installation
============================

This will install Chat TTS Reader on your computer.

FEATURES
--------
* Read YouTube, Kick, and TikTok live chat via text-to-speech
* Automatic Twitch live detection (start when you go live)
* High-quality Microsoft Edge neural voices
* Windows SAPI voices (works offline)
* Customizable message filters
* Block specific users or words
* Runs in background during your stream

REQUIREMENTS
------------
* Windows 10 or Windows 11
* Internet connection (for Edge TTS and chat connections)

AFTER INSTALLATION
------------------
1. Run "Configure" from the Start Menu to set up your platforms
2. Run "Audio Test" to verify TTS is working
3. Run "Chat TTS Reader" to start manually, or enable auto-start

IMPORTANT NOTES
---------------
* YouTube/Kick/TikTok chat only works when you're actually live streaming
* Kick.com may block requests due to anti-bot protection
* TikTok requires you to be live for the connection to work
