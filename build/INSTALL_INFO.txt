Chat TTS Reader v1.1.0 Installation
====================================

This will install Chat TTS Reader on your computer.

FEATURES
--------
* Read YouTube, Kick, and TikTok live chat via text-to-speech
* YouTube auto-detect: Enter your channel once, finds live stream automatically
* Kick browser login: Bypass Cloudflare blocking
* Twitch live detection: Start when you go live (press ENTER to bypass)
* High-quality Microsoft Edge neural voices
* Windows SAPI voices (works offline)
* Customizable message filters

REQUIREMENTS
------------
* Windows 10 or Windows 11
* Internet connection (for Edge TTS and chat connections)

AFTER INSTALLATION
------------------
1. Run "Configure" from the Start Menu to set up your platforms
2. Run "Kick Login" if Kick chat isn't connecting
3. Run "Audio Test" to verify TTS is working
4. Run "Chat TTS Reader" to start

IMPORTANT NOTES
---------------
* YouTube: Use channel mode to auto-detect your live stream each time
* Kick: Use browser login if getting blocked, or set chatroom ID manually
* TikTok: Must be live for connection to work
